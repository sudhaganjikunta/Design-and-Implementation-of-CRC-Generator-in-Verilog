# Design-and-Implementation-of-CRC-Generator-in-Verilog
## CRC Transmitter (crc_tx)
### Inputs:

- clk: Drives sequential logic; **CRC** computation and data output update on rising edge.

- rst: Resets outputs (data_out=0, crc_out=0).

- enable: When high, triggers **CRC** computation and data frame generation.

- data_in[6:0]: 7-bit input data word.

  ---

### Outputs:

- crc_out[2:0]: Computed CRC bits using polynomial  \(x^{3}+1\)

- data_out[9:0]: Concatenation of data_in and crc_out → [data_in (7 bits) + crc_out (3 bits)].

---
### Behavior:

- On reset → outputs cleared.

- On enable → CRC computed, appended to data, and stored in data_out.

---
## CRC Receiver (crc_rx)
### Inputs:

- clk: Synchronizes CRC recomputation.

- rst: Clears CRC output.

- enable: When high, recomputes CRC for received data.

- data_in[6:0]: Received 7-bit data portion.

- crc_received[2:0]: Received CRC bits.

---
### Outputs:

- crc_out[2:0]: CRC recomputed locally from data_in.

- data_valid: High if crc_out == crc_received, else low.

---
### Behavior:

- On reset → crc_out=000.

- On enable → recomputes CRC and compares with received CRC.

- If match → data_valid=1 (data integrity confirmed).

- If mismatch → data_valid=0 (error detected).

---
## Top-Level Testbench (top_tb)
### Inputs (common):

- clk: Shared clock for Tx and Rx.

- rst: Shared reset.

- enable: Controls both Tx and Rx simultaneously.

- tx_data_in[6:0]: Input data word for transmitter.

---
### Outputs:

- tx_crc_out[2:0]: CRC generated by transmitter.

- tx_data_out[9:0]: Frame transmitted (data + CRC).

- rx_crc_out[2:0]: CRC recomputed by receiver.

- data_valid: Indicates whether received CRC matches transmitted CRC.

---
### Behavior:

- Tx computes CRC and appends to data → sends 10-bit frame.

- Rx extracts data and CRC from frame.

- Rx recomputes CRC from data.

- Compares recomputed CRC with received CRC.

- If equal → data_valid=1 (successful transmission).

- If not equal → data_valid=0 (error detected).

---
## Example Signal Flow
```
Input: tx_data_in = 1010110

Tx computes CRC → tx_crc_out = 101

Tx frame → tx_data_out = 1010110101

Rx extracts → data_in = 1010110, crc_received = 101

Rx recomputes CRC → rx_crc_out = 101

Comparison → data_valid = 1 ✅
```
